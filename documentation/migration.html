Migration Tool Manual

1. What is it?

The migration tool gives you the possibility to upgrade to Chamilo 2.0 without losing any data.

1.1 How to start

First you have to decide if you want to migrate with the webinterface or the commandline.

    Webinterface
    1. When you go to the page of the migration tool you have to select the old system you worked with and click next.

    2. Fill in the path where your old system is installed and select which blocks you want to migrate.
       Which blocks you will be able to migrate will depend on if you want to migrate courses.
       Click next

    3. For each block migrated you have to click next to start the migration of the next block
       until you reached the last block.

    Commandline
    1. In the migration folder you have a file called settings.inc.php.

    2. Fill in the name and path of your old system and select which blocks you want
       to migrate with 1(migrate) and 0(don't migrate).
       Which blocks you will be able to migrate will depend on if you want to migrate courses.

    3. If you work with Linux start the migrationtool with ./commandlinemigrationlinux in the migration folder.
       If you work with Windows start the migrationtool with commandlinemigrationwindows in the migration folder.
